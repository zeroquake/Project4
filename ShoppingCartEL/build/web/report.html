Shopping Cart EL Report

Για το δεύτερο κομμάτι της εργασίας δημιουργήσαμε ένα καλάθι αγοράς προιόντων μόνο με τη χρήση Javabeans και EL.

Δεν καταφέραμε να υλοποιήσουμε cookies για την συγκεκριμένη εργασία παρόλη την έρευνά μας στο διαδίκτυο.
Παρ'όλα αυτά διατηρούμε τις τιμες με την βοήθεια των beans καθ'όλη τη διάρκεια του session.

Τις τιμές των προιόντων τις αρχικοποιούμε μέσα στο JSP. Προσπαθήσαμε να κάνουμε parse ένα αρχείο με την βοήθεια του API JSTL,
αλλά παρόλο που καταφέραμε να διαβάσουμε τις τιμές με το tag x:parse ... δεν μπορέσαμε να τις εκχωρίσουμε στα beans
(Οι γνώσεις μας για τα taglibs δεν είναι επαρκείς).

Δεν χρησιμοποιήσαμε servlet αλλά μόνο τις απαραίτητες κλάσεις με τις συναρτήσεις των προιόντων τις οποίες και κάναμε import στο index.jsp.

Οι δυσκολίες ήταν αρκετές μιάς και ήταν οι πρώτη μας φορά που ασχολούμασταν με κάτι τέτοιο.
Παρ'όλα αυτα καταφέραμε να φτιάξουμε κατi λειτουργικό.
